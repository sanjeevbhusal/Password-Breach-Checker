
# Password Breach Checker
This simple project checks if the user's password was breached in any data leaks using API from haveibeenpwned.com

To use it simply clone the repositery and install the required packages available in requirements.txt

Then run the file from command line and pass passwords you wish to check if were breached or not.

Example: python main.py facebook123 hello

Best practises is used while checking the password breach.
